## 2026-07-04 (3): サブタブ表示バグの修正

### やったこと

- **バグ修正**: `.subtabs` の `display: flex` が `hidden` 属性（ブラウザ標準の非表示）を上書きしており、選択中でないカテゴリのチップ行が表示されたままになっていた。`.subtabs[hidden] { display: none; }` を追加して解消（ユーザー報告により発覚）
- CSS/JSの読み込みにバージョン付きURL（`?v=20260704b`）を導入。GitHub Pagesのキャッシュ（最大10分）で古い表示が残る問題への対策。**今後CSS/JSを変更したらこのバージョン文字列も更新すること**

### 教訓

- `hidden` 属性は要素にCSSで `display` を指定すると効かなくなる。`[hidden] { display: none }` の明示が必要
- 表示状態の検証はJSのプロパティ（`box.hidden`）だけでなく `getComputedStyle().display` で見た目も確認する
